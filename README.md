# Social media app created with T3 stack
App with layout similar to Twitter, allowing to post emojis.

## Usage
+ Visit app deployed on Vercel [here](https://t3-stack-testing-neua.vercel.app/)
+ Sign in with GitHub account
+ Enjoy the app
